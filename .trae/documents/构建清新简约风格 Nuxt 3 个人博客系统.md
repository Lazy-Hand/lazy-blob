# 开发计划：清新简约风格 Nuxt 3 个人博客

## 技术栈

* **前端**: Nuxt 3 + Vue 3 + TypeScript

* **样式**: Tailwind CSS + Nuxt UI

* **图标**: Lucide Icons

* **数据**: Mock 数据（预留 NestJS 后端接口）

***

现有的配置不要去动，页面可以使用 @nuxt/ui和shadcn-nuxt这两个ui加速开发

<br />

第一阶段：项目基础架构搭建

### 1.1 创建目录结构

```
app/
├── assets/
│   ├── data/           # Mock 数据
│   ├── images/         # 图片资源
│   └── css/           # 样式文件
├── components/
│   ├── layout/         # 布局组件
│   ├── feature/        # 功能组件
│   └── ui/            # UI 组件（已存在）
├── pages/
│   ├── index.vue       # 首页
│   ├── blog/          # 博客相关
│   ├── archive/       # 归档
│   ├── treehole/      # 树洞
│   ├── photography/   # 摄影
│   └── about/        # 关于
└── composables/      # 组合式函数
```

### 1.2 创建 Mock 数据结构

* 博客文章数据

* 归档数据

* 树洞（日记）数据

* 摄影作品数据

* 友情链接数据

* 成长时间线数据

### 1.3 更新应用配置

* 修改站点标题、描述、关键词

* 配置主题颜色（清新绿色系）

* 设置 SEO 元数据
